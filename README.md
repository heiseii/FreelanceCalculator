# 🧮 Freelance Calculator

> Calculadora web pensada para **freelancers argentinos** que necesitan calcular sus tarifas y honorarios de forma rápida y sencilla.

🌐 **Demo en vivo:** [freelance-calculator-seven.vercel.app](https://freelance-calculator-seven.vercel.app)

---

## ✨ ¿Qué hace?

Una herramienta práctica para freelancers en Argentina que ayuda a:

- Calcular el precio de un proyecto o servicio en pesos argentinos
- Estimar honorarios por hora de trabajo
- Tener en cuenta factores como impuestos, inflación y márgenes de ganancia

## 🚀 Cómo correrlo localmente

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/heiseii/FreelanceCalculator.git
   ```

2. Abrir el archivo `index.html` directamente en tu navegador, o usar un servidor local:
   ```bash
   # Con Python (recomendado)
   python -m http.server 8000
   ```

3. Entrar a `http://localhost:8000` en tu navegador.

> No requiere instalación de dependencias.
